# Portal Integrado OVA-TGS

Repositorio de la asignatura **Teoría General de Sistemas** — Universidad Simón Bolívar, Campus Cúcuta.

## Portal web

Abrir `index.html` en la raíz para acceder al portal integrado (dashboard, 14 OVAs embebidas, Centro de Aprendizaje, tema claro/oscuro).

## Estructura

- `index.html` — Portal principal (gemelo del Portal de Compiladores)
- `OVAS/` — Objetos Virtuales de Aprendizaje originales
- `shared/` — Design system USB (`usb-ova-tokens.css`, `usb-ova-components.css`, `usb-ova-theme.js`)
- `js/` — Centro de Aprendizaje
- `INFORME-AUDITORIA.md` — Auditoría Fase 1
- `INFORME-FINAL.md` — Validación Fase 12

## GitHub Pages

Incluye `.nojekyll`. Publicar la raíz del repositorio como sitio estático.
